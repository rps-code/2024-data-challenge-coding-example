# 2024 Data Challenge Coding Example

A small project visualising some data using Vue 3 and Tailwind.

## Frontend Stack

-   Vue 3 (Composition API)
-   Tailwind
-   Vite
-   Typescript

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
