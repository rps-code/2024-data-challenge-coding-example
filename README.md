# Fancensus Coding Example

Thank you for the opportunity to do this, it was a fun project to do!

## Frontend Stack

-   Vue 3 (Composition API)
-   Tailwind
-   Vite
-   Typescript

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
